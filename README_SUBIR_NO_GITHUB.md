## Como subir no GitHub

1. Abra o repositório `os-bichinhos-cantam`.
2. Faça upload/substituição destes arquivos na raiz:
   - `index.html`
   - `styles.css`
   - `favicon.svg`
   - `robots.txt`
   - `sitemap.xml`
3. Faça upload/substituição da pasta:
   - `assets/`
4. Faça commit.

## Arquivos visuais gerados
